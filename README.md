# 摩估 MotoValue · 二手摩托车科学估价（MVP）

面向中国大陆的二手摩托车价格评估平台。第一版聚焦**估价引擎**：输入车型与车况，输出**可量化、可解释、符合国情**的价格区间。

## 快速开始

```bash
npm install
npm start            # 启动 http://localhost:3000
```

开启定时抓取官方价（可选）：

```bash
ENABLE_SCRAPER=1 SCRAPE_ON_START=1 npm start
npm run scrape       # 或手动跑一轮抓取
```

## 估价模型 v2（符合国情的量化公式）

```
估价 = 新车指导价
     × 综合残值( 年限基础残值 ⊗ 类别折旧 ⊗ 品牌保值 )
     × 里程修正(按车型类别校准年均里程)
     × 车况系数(外观·机械·事故·改装·保养)
     × 过户记录(一手/过户次数)
     × 排放系数(国四/国三/国二)
     × 手续系数(三证/发票/登记证/抵押)
     × 区域系数(禁摩/限摩/不禁摩)
```

**残值叠加法**：`残值 = 1 - (1 - 年限基础残值) × 类别折旧 × 品牌折旧`。新车时缺口为 0、不受类别/品牌影响（符合“新车都接近指导价”）；年限越久缺口越大、类别与品牌差异随之放大（符合“老车才看出保值差异”）。

针对性纳入的**中国大陆国情变量**：
- **13 年强制报废**：主导折旧曲线，临近报废残值快速归零并触发风险提示。
- **车型类别差异化折旧**：踏板/弯梁（代步）贬值快，复古/拉力（玩乐）保值，越野损耗大。
- **品牌保值梯度**：本田 > 雅马哈 > 铃木/豪爵 > 川崎/哈雷 > 春风 > 钱江/无极 > 凯越（具体系数可调）。
- **过户次数 / 是否一手**：一手未过户加价，多次过户明显掉价。
- **排放标准**：国三车难过户/迁入，显著折价。
- **手续完整度**：缺登记证书（大绿本）无法过户，重大缺陷。
- **禁摩/限摩政策**：影响本地需求与流通性。

所有系数集中在 `server/valuation.js`（类别折旧 `CATEGORY_DECAY`、品牌折旧 `BRAND_DECAY`、过户 `TRANSFER_FACTOR` 等均可单独调），估价结果附**逐项明细**，用户能看到“为什么是这个价”。

## 目录结构

```
server/
  index.js              Express 入口 + API
  valuation.js          估价引擎(核心算法)
  db.js                 JSON 数据存储(种子→运行时)
  scraper/
    index.js            抓取调度
    scheduler.js        node-cron 定时
    run-once.js         手动抓取 CLI
    adapters/           各品牌抓取适配器(可插拔)
data/
  models.seed.json      种子车型库(新车指导价 + 来源 + 更新日期)
  models.json           运行时库(抓取回写, 不入版本库)
public/                 前端页面
```

## 价格数据与抓取架构

种子库中的新车指导价来自公开检索，`priceVerified: false` 表示需人工核对或由抓取任务刷新。抓取为“尽力而为”：成功则更新并标记已核实，失败则保留库存价，网站始终可用。

**抓取来源分层（`server/scraper/`）：**
- **主源 摩托范 58moto.com**（`adapters/moto58.js`）：聚合站覆盖全品牌、结构统一、有清晰“指导价”字段，作为权威源**优先执行**。为车型补 `ref` 字段（该车在摩托范的页面链接）即纳入主源抓取。
- **兜底 各 OEM 官网**（`adapters/{haojue,cfmoto,qjmotor,voge,yamaha,honda}.js`）：仅补主源未命中的车型，按车型 `source`（官网链接）抓取。
- 价格提取 `extractPriceCNY` 优先取“指导价”旁数字，否则在候选中取众数，规避促销价/配件价干扰（含单元测试用例）。

**已知限制（实测）：** 国内站点（含 58moto、各 OEM 官网）对非本地网络普遍超时/反爬。本机实测仅春风官网可达。**需部署在能访问这些站点的服务器（建议国内机房）抓取才稳定生效**；这也契合“符合本地国情”。`SCRAPE_TIMEOUT_MS` / `SCRAPE_RETRIES` 可调超时与重试。

> 扩展方式：为更多车型在 `data/models.seed.json` 补 `ref`（摩托范链接），即可被主源抓取刷新。

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET  | `/api/brands` | 品牌列表 |
| GET  | `/api/models?brand=` | 车型列表(可按品牌过滤) |
| GET  | `/api/options` | 估价表单可选项 |
| POST | `/api/valuation` | 估价(body: modelId + 车况字段) |

## 交易功能（第二阶段，已完成 MVP）

个人可发布、浏览、收藏二手车源；发布时复用估价引擎生成**估价快照**，买家在详情页看到「卖家定价 vs 科学估价」的对比与可解释明细。

页面：`/index.html` 估价 · `/market.html` 市场 · `/sell.html` 发布 · `/listing.html?id=` 详情。
数据：`data/listings.json`（运行时生成，不入版本库）。身份为客户端 `ownerToken`（localStorage，无密码 MVP），收藏存 localStorage。

新增 API：

| 方法 | 路径 | 说明 |
|------|------|------|
| GET    | `/api/listings?brand=&category=&maxPrice=&ownerToken=` | 车源列表(可筛选) |
| GET    | `/api/listings/:id` | 车源详情 |
| POST   | `/api/listings` | 发布(服务端生成估价快照) |
| DELETE | `/api/listings/:id` | 下架(需 ownerToken) |
